setuptools
===============
- 名前空間パッケージ
- モジュールとかをまとめてるディレクトリのこと = 名前空間
- 同じパッケージ以下のモジュールを複数の配布物にする仕組み
- ``__init__`` をあえておかないでやる
- native になると、 ``__init__`` なくなる
- 名前空間のはややこしいよ


pypi.org
===============
- pip と virtualenv 
- pip は単なるインストーラー
- pip


pipenv
================
- 遅い
- virtualenv を外から管理する

- 一番最初に python のバージョンを明示的に指定したほうがよい
- 裏側には virtualenv がある
- activate しなくてもできる ``run``
- ロックファイル
- dev_dependensies ある

- 新しい
